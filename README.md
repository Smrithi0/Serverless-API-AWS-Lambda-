Project Name: Serverless API (AWS Lambda)

Project Scope: This project involves developing a serverless REST API using AWS Lambda and Amazon API Gateway. The implementation includes creating a Lambda function using Python, configuring an HTTP API in API Gateway, defining multiple API routes, integrating the routes with a single Lambda function, and testing the endpoints using the generated Invoke URL. The API provides employee, department, company, and health information through structured JSON responses, demonstrating serverless application development and RESTful API implementation on the AWS cloud platform.
